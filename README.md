# React app

My First React App
react-app is simple app that works with mongodb app.

# Running Locally

1. git clone react-app
2. git clone mongoServer
3. cd mongoServer
4. npm install
5. npm start
6. cd react-app
7. npm install
8. npm start
9. Open http://localhost:3000 and take look around.

- react
